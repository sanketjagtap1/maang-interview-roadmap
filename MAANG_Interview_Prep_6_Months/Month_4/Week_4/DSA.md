# DSA - Week 4

## Topics (Order)
1. Mock Interview Practice
2. Company-Specific Problems
3. Hard Problem Practice
4. Speed and Accuracy

## Patterns to Learn
- Problem identification (quick pattern recognition)
- Solution optimization (time/space trade-offs)
- Code quality (clean, readable, maintainable)
- Communication (explain while coding)

## Problems (Focus on Quality)

### Mock Interviews
- Practice 2-3 problems per day (45-60 min each)
- Mix of easy, medium, hard problems
- Random topics from all months
- Time-boxed practice

### Company-Specific
- Research target company's favorite problems
- Practice problems from company's interview experiences
- Focus on company's preferred patterns

### Hard Problems
- [GOOD] Trapping Rain Water II (407)
- [GOOD] Serialize and Deserialize N-ary Tree (428)
- [GOOD] Design In-Memory File System (588)
- [GOOD] Word Ladder II (126)

## Focus
- Speed: Solve medium in 25-30 min
- Accuracy: Handle all edge cases
- Communication: Explain approach clearly
- Code quality: Production-ready code
