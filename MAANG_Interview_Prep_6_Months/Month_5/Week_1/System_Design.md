# System Design - Week 1

## Core Concepts
- Mock System Design Interviews
- Company-Specific Design Focus
- Design Deep Dives
- Trade-off Analysis

## Interview Focus Points
- **Mock Interviews**: 2-3 system design mocks per week
- **Company Focus**: Research target company's architecture
- **Deep Dives**: Handle detailed technical questions
- **Trade-offs**: Explain decisions clearly

## Practice
- Mock system design interviews (45 min each)
- Practice common system design problems
- Review real-world architectures
- Practice explaining trade-offs

## Common Design Problems
- Design Twitter (feed, timeline)
- Design Instagram (photo sharing)
- Design Uber (ride matching)
- Design Netflix (video streaming)
- Design WhatsApp (messaging)

## Reading
- Company engineering blogs
- Real-world system architectures
- System design case studies
- Design patterns and best practices
