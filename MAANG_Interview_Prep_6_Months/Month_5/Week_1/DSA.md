# DSA - Week 1

## Topics (Order)
1. Mock Interview Intensive
2. Company-Specific Problem Sets
3. Hard Problem Mastery
4. Speed Drills

## Focus
- **Mock Interviews**: 3-4 per week (45-60 min each)
- **Company Focus**: Target company problem sets
- **Hard Problems**: Master complex problems
- **Speed**: Medium problems in 20-25 min

## Problems (Practice-Based)

### Daily Practice
- 2-3 problems per day (mix of easy/medium/hard)
- Time-boxed: Easy (15 min), Medium (25 min), Hard (40 min)
- Random topics from all months
- Focus on accuracy and speed

### Company-Specific
- Google: Graph, DP, String problems
- Amazon: Array, String, Tree problems
- Meta: Array, Graph, Backtracking
- Microsoft: Tree, Graph, DP problems
- Apple: Array, String, Design problems

### Hard Problems
- [GOOD] Trapping Rain Water II (407)
- [GOOD] Word Ladder II (126)
- [GOOD] Serialize and Deserialize N-ary Tree (428)
- [GOOD] Design In-Memory File System (588)
- [GOOD] Alien Dictionary (269)

## Focus
- Pattern recognition: Identify problem type in 2-3 min
- Solution quality: Clean, optimized code
- Communication: Explain approach clearly
- Edge cases: Handle all edge cases
