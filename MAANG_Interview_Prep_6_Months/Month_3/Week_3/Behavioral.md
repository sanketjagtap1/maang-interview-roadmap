# Behavioral - Week 3

## STAR Stories to Prepare

### Data Processing / Analytics
- **Situation**: Large-scale data processing challenge
- **Task**: Process, analyze, or transform large datasets
- **Action**: Data pipeline design, optimization, tool selection
- **Result**: Processing time reduction, insights delivered

### Search / Recommendation
- **Situation**: Search or recommendation system project
- **Task**: Build or improve search/recommendation quality
- **Action**: Algorithm selection, indexing, ranking optimization
- **Result**: Relevance improvement, user engagement metrics

### Real-time Systems
- **Situation**: Real-time data processing requirement
- **Task**: Build low-latency data processing system
- **Action**: Stream processing, architecture design, optimization
- **Result**: Latency reduction, throughput increase

## Question Themes
- "Tell me about a time you..."
  - Worked with large datasets
  - Built a search or recommendation feature
  - Optimized data processing performance
  - Worked on real-time systems
  - Made data-driven decisions

## Preparation
- Add data/analytics stories
- Practice explaining algorithms simply
- Prepare examples of performance optimizations
- Research company's data challenges
