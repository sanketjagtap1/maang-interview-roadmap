# Behavioral - Week 1

## STAR Stories to Prepare

### Optimization / Performance
- **Situation**: Performance bottleneck or optimization opportunity
- **Task**: Optimize system performance
- **Action**: Profiling, bottleneck identification, optimization
- **Result**: Latency reduction, throughput increase, cost savings

### Database / Data
- **Situation**: Database performance or data challenge
- **Task**: Optimize database or handle data at scale
- **Action**: Query optimization, indexing, data modeling
- **Result**: Query time reduction, scalability improvement

### Monitoring / Observability
- **Situation**: Need for better monitoring or observability
- **Task**: Implement monitoring/observability solution
- **Action**: Tool selection, metric definition, alerting setup
- **Result**: Visibility improvement, faster incident detection

## Question Themes
- "Tell me about a time you..."
  - Optimized database performance
  - Implemented monitoring or observability
  - Improved system performance significantly
  - Handled a data migration
  - Optimized query performance

## Preparation
- Add optimization/performance stories
- Practice explaining technical optimizations
- Prepare examples of performance improvements
- Research company's performance challenges
